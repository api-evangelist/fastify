<figure><img alt="" src="https://cdn-images-1.medium.com/max/1000/0*bwQSF6VqFuZ88-FN" /><figcaption>Courtesy of <a href="https://unsplash.com/@arnaudgillard">Arnaud Gillard</a></figcaption></figure><p>We are thrilled to announce the final release of the third major version of Fastify! In April this year, we published the first Fastify v3 Release Candidate Since then we have been focusing our efforts on porting all modules maintained in the Fastify GitHub organization to Fastify v3. We would like to thank our amazing community for all of the work they put in to making this a reality.</p><p>This new version of Fastify includes some changes that will allow you to have an even better experience with Fastify, starting from the brand new type definitions, the new full support of Express middleware, and the refactored request validation.</p><p>Full release notes can be found <a href="https://github.com/fastify/fastify/blob/master/docs/Migration-Guide-V3.md">here</a>, with a handy guide to help you migrate from v2.</p><p>The most noticeable breaking changes are:</p><ul><li>Run an Express application inside Fastify (<a href="https://github.com/fastify/fastify/blob/master/docs/Migration-Guide-V3.md#changed-middleware-support-2014">#2014</a> and <a href="https://github.com/fastify/fastify-express">fastify-express</a>)</li><li>Changed logging serialization (<a href="https://github.com/fastify/fastify/blob/master/docs/Migration-Guide-V3.md#changed-logging-serialization-2017">#2017</a>)</li><li>Changed schema substitution (<a href="https://github.com/fastify/fastify/blob/master/docs/Migration-Guide-V3.md#changed-schema-validation-options-2023">#2023</a>)</li><li>Improved TypeScript support (<a href="https://github.com/fastify/fastify/blob/master/docs/Migration-Guide-V3.md#changed-typescript-support">description</a>)</li></ul><p>Fastify v3 can now be installed with npm install fastify.</p><p>Please let us know about any compatibility and upgrade issues at <a href="https://github.com/fastify/help">fastify/help</a>!</p><h3>Updated benchmarks</h3><a href="https://medium.com/media/a840f10fdcf75461ea7fd2d9078dfa1b/href">https://medium.com/media/a840f10fdcf75461ea7fd2d9078dfa1b/href</a><h3>Acknowledgments</h3><ul><li>More than 2300 commits</li><li>More than 130 releases,</li><li>More than 1000 forks</li><li>Used in production by more than 20 companies</li><li>More than 540k downloads per month</li><li>136 officially recognized plugins.</li></ul><p>This wouldn’t have been possible without the help of our fantastic team, which we would like to thank for all their hard work:</p><p><a
